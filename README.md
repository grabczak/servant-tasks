# Servant Tasks

Backend API made with Haskell and [Servant](https://docs.servant.dev/). Features JWT authentication and database persistence.

## Quick Start

Use [GHCup](https://www.haskell.org/ghcup/) to install GHC, Stack and Cabal.
Then clone the repo and run:

```bash
$ stack build
$ stack run
```

## API Documentation

Use [Insomnia](https://insomnia.rest/) to explore API endpoints. Import `Insomnia.yaml` to get started.
